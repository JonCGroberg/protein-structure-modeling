## Amino acids

### Mutation syntax

```python
original_amino_acid = 'A'
location = 201
mutant_amino_acid = "K"

f"{original_amino_acid}{location}{mutant_amino_acid}"

# Result will be A201K
```

#### Examples

- K417N
- E484K
- N501Y
